# NLP Sentiment Analysis using BERT

This project implements sentiment classification using the BERT transformer model.

## Dataset
IMDB Movie Review Dataset

## Technologies
- Python
- PyTorch
- HuggingFace Transformers
- Google Colab

## Model
Pretrained BERT (bert-base-uncased) fine-tuned for binary sentiment classification.

## Pipeline
Text → Tokenization → BERT Model → Sentiment Classification

## Example

Input:
"This movie was fantastic"

Output:
Positive Review
